# Town of Harpy
### By CorranEveryone
This mod is developed to add on to Wathe, making the game have a funner gameplay loop by adding more features.

## Contents
### Gamemodes
- Murder Mystery *(wathe:murder)*
- Loose Ends *(wathe:loose_ends)*
- Discovery/Exploration Mode *(wathe:discovery)*
### Roles
- Civilian
    - Survive
- Vigilante
    - Shoot the killer
- Killer
    - Kill everyone aboard the train

## Fun Facts
- The name **Town of Harpy** is inspired by [Town of Us](https://github.com/AU-Avengers/TOU-Mira) which is a mod for [Among Us](https://www.innersloth.com/games/among-us/), another Murder Mystery game.
